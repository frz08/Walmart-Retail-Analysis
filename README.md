# 🛒 Walmart Retail Dataset Analysis

An end-to-end retail data analysis project covering data cleaning, exploratory analysis, and regression modeling using SQL and Excel.

## 📌 Overview

This project works through the full analytics pipeline on a Walmart retail dataset — from raw data cleaning in SQL through to statistical analysis and visual reporting in Excel.

## 🔍 What's Inside

- **Data Cleaning** — SQL scripts to handle missing values, duplicates, and data type formatting
- **Exploratory Analysis** — Pivot Tables summarizing sales trends, regional performance, and product categories
- **Visual Reporting** — Pivot Charts for clear visual communication of key metrics
- **Regression Analytics** — Statistical regression analysis using Excel's Data Analysis Toolpak to model sales drivers

## 🛠️ Tools Used

| Tool | Purpose |
|------|---------|
| SQL | Data cleaning and transformation |
| Excel | Analysis, Pivot Tables, Pivot Charts |
| Data Analysis Toolpak | Regression modeling |
